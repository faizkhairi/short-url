# Screenshot Capture Instructions

## What to Capture: Analytics Dashboard

**Type**: Static Screenshot (PNG)
**Dimensions**: 800-1200px wide
**Quality**: High (PNG format)

---

## Steps to Capture

1. **Open the app**: https://short-url-livid-six.vercel.app

2. **Create a few test URLs**:
   - Enter a URL (e.g., https://github.com/faizkhairi)
   - Click "Shorten"
   - Repeat 2-3 times

3. **Click on a short link** to generate some click data:
   - Open the short URL in a new tab (e.g., https://short-url-livid-six.vercel.app/abc123)
   - Click it from different devices/browsers if possible (to get device variety)
   - Visit 5-10 times to generate analytics data

4. **Navigate to the dashboard** for one of your URLs

5. **Set browser to consistent size**:
   - Open DevTools (F12)
   - Click "Toggle device toolbar" icon
   - Select "Laptop with HiDPI screen" or similar preset

6. **Take screenshot**:
   - Capture the full analytics view showing:
     - Click count
     - Charts (Clicks over time, Device breakdown, Top referrers)
     - Recent clicks table
   - Use Windows Snipping Tool, macOS Screenshot, or browser screenshot extension

7. **Save the screenshot**:
   - Format: PNG
   - File name: `dashboard.png`
   - Location: `/c/Repo/short-url/docs/dashboard.png`

8. **Optional**: Crop to remove browser chrome (address bar, bookmarks)

---

## Tips

- Make sure the charts have data (not empty states)
- Screenshot should show at least 3-5 clicks with varied data
- Keep the layout clean and professional
